## Main project — CIRI_WEB

**CIRI_WEB** is a private internal operations platform that replaces fragmented
desktop workflows with a single web interface for support and monitoring teams.

The application is designed as a modular monolith with clear boundaries between
the HTTP/runtime layer and domain logic.

### What it includes

- Incident and planned-work workflows
- Operational preflight checks
- Background log-export jobs
- Alert classification workers
- Reference-data management
- External service integrations
- Health and readiness endpoints
- Safe dry-run and production execution modes
- Idempotent operations and structured error handling

### Architecture

```text
Static HTML/CSS/JavaScript frontend
                ↓
            FastAPI API
                ↓
       Services and domain logic
                ↓
PostgreSQL · background workers · external integrations
```
